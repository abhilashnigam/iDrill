iDrill

iOS Static analyser

iDrill

This script allows you to perform static analysis of an iOS Application. This allows the user to perform static analysis on the iDevice itself and reduces the dependency of MacOS to nill.

Note: The tool does not read the code of the application for that you still need a MacOS and can export the IPA for analysis.

Usage:

  • Add the source in Cydia using the link below
  • Install iDrill-v2
  • SSH to the device ssh root@
  • Run iDrill
